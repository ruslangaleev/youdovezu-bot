# Настройка Яндекс.Карт для автодополнения адресов

## Получение API ключа

1. Перейдите на [Яндекс.Разработчик](https://developer.tech.yandex.ru/)
2. Войдите в аккаунт Яндекс
3. Создайте новый проект или выберите существующий
4. В разделе "API" выберите "Яндекс.Карты"
5. Скопируйте полученный API ключ

## Настройка в проекте

### Вариант 1: Переменная окружения (рекомендуется)

1. Создайте файл `.env` в папке `frontend/`:
```env
REACT_APP_YANDEX_API_KEY=ваш_api_ключ_здесь
```

2. Перезапустите приложение

### Вариант 2: Прямое указание в коде

1. Откройте файл `frontend/src/yandex-config.ts`
2. Замените `YOUR_YANDEX_MAPS_API_KEY` на ваш реальный API ключ:
```typescript
API_KEY: 'ваш_api_ключ_здесь',
```

## Ограничения и настройки

- Поиск ограничен территорией Башкортостана
- Максимум 5 результатов в подсказках
- Автодополнение срабатывает при вводе от 3 символов

## Тестирование

1. Запустите приложение
2. Перейдите в "Ищу машину" → "Новая поездка"
3. Начните вводить адрес в поле "Откуда" или "Куда"
4. Должны появиться подсказки от Яндекс.Карт

## Возможные проблемы

- **Ошибка загрузки**: Проверьте правильность API ключа
- **Нет подсказок**: Убедитесь, что вводите минимум 3 символа
- **Подсказки не закрываются**: Проверьте обработчики событий в коде
