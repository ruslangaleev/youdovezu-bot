# YouDovezu Frontend

React приложение для P2P платформы организации поездок в Башкирии.

## Описание

YouDovezu - это веб-приложение для поиска и создания поездок между городами Башкирии. Пользователи могут находить попутчиков или предлагать свои поездки.

## Технологии

- **React 19** - основной фреймворк
- **TypeScript** - типизация
- **CSS3** - стилизация

## Запуск

### Требования
- Node.js 16+
- npm или yarn

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm start
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

### Сборка для продакшена
```bash
npm run build
```

### Запуск тестов
```bash
npm test
```
