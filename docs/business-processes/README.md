## Общее описание

**YouDovezu** — это P2P-сервис для организации совместных поездок в Республике Башкортостан. Платформа объединяет людей, которые едут в одном направлении, позволяя им делиться поездкой и расходами. Это не агрегатор перевозок, а сервис для поиска попутчиков.

**Основные функции:**
- Регистрация и авторизация пользователей через Telegram
- Публикация запросов на поездки с указанием маршрута
- Просмотр списка своих запросов на поездки
- Редактирование существующих запросов
- Удаление запросов
- Загрузка документов для получения доступа к просмотру запросов других пользователей
- Предложение услуг с указанием цены
- Организация совместных поездок

**Роли в системе:**
- **Offerer** — пользователь, который предоставляет автомобиль
- **Requester** — пользователь, который ищет место в попутном авто
