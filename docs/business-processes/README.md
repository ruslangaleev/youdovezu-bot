Данная папка содержит описание всех бизнес-процессов приложения YouDovezu.

1. [Регистрация пользователя](./user-registration.md) - Процесс регистрации и авторизации через Telegram

3. [Водительский режим](./driver-mode.md) - Получение доступа к водительскому режиму через загрузку документов

## Общее описание

**YouDovezu** — это платформа для поиска попутчиков и организации поездок в Республике Башкортостан. Пользователи могут создавать поездки, указывая маршрут от пункта отправления до пункта назначения, и находить попутчиков для совместных поездок.

**Основные функции:**
- Регистрация и авторизация пользователей через Telegram
- Создание поездок с указанием маршрута
- Просмотр списка своих поездок
- Редактирование существующих поездок
- Удаление поездок
- Загрузка документов для получения доступа к водительскому режиму
